A small library of PHP helper functions

- Date formatter
- String formatter
- Number formatter
- Slug generator
- Pagination generator
- Regex to date, mail, CPF, CNPJ, links, MD5, color, etc.
- Http Response Code
- Mime Content Type
- IPv4 and IPv6 convert to binary, hexadecimal
- Password generator
- String PAD
- Leap year
- Detect URL and Add <a> to text

- Others
